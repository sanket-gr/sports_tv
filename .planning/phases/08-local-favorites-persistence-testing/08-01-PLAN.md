## Proposed Changes

### Mobile App Module

---

#### [NEW] [ic_star.xml](file:///d:/projects/sports_tv/android_tv_app/mobile/src/main/res/drawable/ic_star.xml)
- Vector drawable for a filled star icon (yellow/gold color).

#### [NEW] [ic_star_border.xml](file:///d:/projects/sports_tv/android_tv_app/mobile/src/main/res/drawable/ic_star_border.xml)
- Vector drawable for an empty star outline icon (white/gray color).

#### [NEW] [FavoritesManager.kt](file:///d:/projects/sports_tv/android_tv_app/mobile/src/main/java/com/sportstv/mobile/FavoritesManager.kt)
- Create a utility object `FavoritesManager` using `SharedPreferences` to load, store, check, and toggle favorite stream IDs.

#### [NEW] [bottom_sheet_stream_detail.xml](file:///d:/projects/sports_tv/android_tv_app/mobile/src/main/res/layout/bottom_sheet_stream_detail.xml)
- A modern bottom sheet layout with:
  - Stream title, category name, and sport badges.
  - Large thumbnail preview.
  - A toggle button for adding/removing from favorites (using star drawables).
  - A "Watch Live" play button.

#### [NEW] [StreamDetailBottomSheet.kt](file:///d:/projects/sports_tv/android_tv_app/mobile/src/main/java/com/sportstv/mobile/StreamDetailBottomSheet.kt)
- Create `StreamDetailBottomSheet` subclassing `BottomSheetDialogFragment`.
- Load selected stream arguments, display info, and manage favorite button states via `FavoritesManager`.
- Launch `PlaybackActivity` and dismiss the dialog on "Watch Live" click.
- Notify parent view on favorites change.

#### [MODIFY] [StreamAdapter.kt](file:///d:/projects/sports_tv/android_tv_app/mobile/src/main/java/com/sportstv/mobile/StreamAdapter.kt)
- Check `FavoritesManager` in `onBindViewHolder` to display the favorite star badge (`tv_favorite_star`) if the stream is favorited.

#### [MODIFY] [MainActivity.kt](file:///d:/projects/sports_tv/android_tv_app/mobile/src/main/java/com/sportstv/mobile/MainActivity.kt)
- Change stream click callback to open `StreamDetailBottomSheet` instead of `PlaybackActivity`.
- In `showFavoritesView()`, filter `allStreams` list using `FavoritesManager.getFavoriteIds()` and bind to adapter.
- In `showHomeView()`, show all streams.
- Refresh list datasets whenever the bottom sheet favorites state is modified.
